# Praktikum Manajemen User MariaDB/MySQL

## Deskripsi
Repository ini berisi hasil praktikum mengenai manajemen user pada MariaDB/MySQL menggunakan Command Prompt (CMD).

## Materi Praktikum
- Login ke MariaDB
- Melihat struktur tabel `user`
- Mengubah password user
- Menerapkan perubahan privilege
- Menampilkan daftar user pada MariaDB
